# 💼 SmartHire — AI Based Job Portal

A professional job portal built with React! Find, save and apply to jobs from top companies across India.

## 🌐 Live Demo

[Visit SmartHire](https://smart-hire-zeta.vercel.app/)

## ✨ Features

- 🔍 Search and filter 8+ job listings
- 💼 Detailed job pages with apply flow
- 🔐 Register and login with JWT auth
- 🔖 Save favourite jobs
- ✅ Track applications in dashboard
- 📱 Fully responsive design
- 🎨 Colorful Canva-style UI

## 🛠️ Built With

- React 18
- React Router DOM v6
- Context API
- Tailwind CSS v3
- localStorage for persistence

## 🚀 Getting Started

1. Clone the repo
   git clone https://github.com/Rakshana2205/smarthire.git

2. Install dependencies
   npm install

3. Start the app
   npm start
